# projectTPE News

## projectTPE 0.1.0 (2025)

### New Features
* Initial release of projectTPE
* EFA module: parallel analysis, KMO, Bartlett, oblimin rotation
* CFA module: MLR estimation, FIML, modification indices
* Longitudinal measurement invariance testing (configural → metric → scalar)
* Latent Profile Analysis (LPA) via tidyLPA
* Built-in simulated Rapor Pendidikan dataset (4 waves, 6 factors)
* Shiny dashboard with 6 panels: Data Input, EFA, CFA, Invariance, LPA, Export
* Export results to multi-sheet Excel workbook
* Support for Rapor Pendidikan data (Paket C and PPS Ulya equivalency schools)
